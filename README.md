# Recipes-APP-ContextAPI
A complete recipe App, made with React, Router, Contex API and BootStrap; You must use npm install in order to run the app

App de receitas, utilizando React: Hooks e Context API
Possível ver, buscar, filtrar, favoritar e acompanhar o processo de preparação de receitas e drinks!
A base de dados serão 2 APIs distintas, uma para comidas e outra para bebidas.
